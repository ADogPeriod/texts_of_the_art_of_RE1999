## 「暴雨」 / THE STORM

### 暴雨症候

**暴雨症候**是一种在「暴雨」发生前产生的异常现象，其对人事物都产生作用，往往伴随着将人变得疯狂的症状。暴雨症候的外在病理特征往往和该时代的艺术或重大事件有关。

*世界的秩序在「暴雨」冲刷之下难以辨认，一切都开始趋于疯狂，走向陌生。生吞黄金，痛饮石油，歌舞于炙热的铁板之上。病人们总是不知道自己病了，任何时候都一样。*

1. 暴雨症候会改变受害者的物理特征，有时则会改变受害者认知，没有统一的病症。
2. 暴雨症候的效果不仅会作用于人，对环境也会造成影响，这种影响会在物理层面改变环境的成分。
3. 暴雨症候在特定情况下，对神秘学家一样可以产生作用。
