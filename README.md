## 「暴雨」 / THE STORM

- 一种时空灾害现象，表现为从天降雨后雨水倒流回空中，并出现整个「时代」的「回溯」。「暴雨」后世界会处于一个与此前不同的「时代」。

### 暴雨症候 / Storm Syndrome

**暴雨症候**是一种在「暴雨」发生前产生的异常现象，其对人事物都产生作用，往往伴随着将人变得疯狂的症状。暴雨症候的外在病理特征往往和该时代的艺术或重大事件有关。  
*世界的秩序在「暴雨」冲刷之下难以辨认，一切都开始趋于疯狂，走向陌生。生吞黄金，痛饮石油，歌舞于炙热的铁板之上。病人们总是不知道自己病了，任何时候都一样。*

1. 暴雨症候会改变受害者的物理特征，有时则会改变受害者认知，没有统一的病症。
2. 暴雨症候的效果不仅会作用于人，对环境也会造成影响，这种影响会在物理层面改变环境的成分。
3. 暴雨症候在特定情况下，对神秘学家一样可以产生作用。
4. 暴雨症候不会立刻扩散到全世界，而是会从「暴雨」临界点开始，随着临近「暴雨」而扩大范围并加剧。越接近临界点症状越严重。同次症候造成的异常行为会因地区文化有细微区别，但物理上的表现形式除了程度基本相同。

### 非对称核素R / Asymmetrical Nuclide R

在「暴雨」雨滴中发现的一种成分，出现后会很快消失，处于不稳定状态。

1. 最早的非对称核素R样本由12岁时的维尔汀在第三次暴雨期间采集，从暴雨雨滴中获得。
2. 非对称核素R在暴雨开始前24小时普遍存在于大气中，此时大气中会产生庞大的神秘学能量，成为一个巨型术式。这种充满灵性的环境可以从外在突破施术条件中血脉的限制，让神秘学家释放更高阶的神秘术。
3. 直接接触不稳定状态的非对称核素R会出现毛发与皮肤组织的受损、变形等症状。
4. 使用声波发生器按固定频率促进非对称核素R发生反应，可在一定程度上模拟「暴雨」环境。

# 组织 / ORGANIZATION

## 鸽子屋 / THE PAX HOUSE

### 「鸽子屋」 / "The Pax House"

——全名为人类和平安全理事会。  
*你一定听说过那高悬于顶的雪白石屋。课本里告诉我们，那是人类和平安全理事会，现今地球最高层管理机构。他们并不直接对人类社会进行管理，只在关乎存亡和发展的重要时刻显现。*

### 白鸽

鸽子屋似乎有使用鸽子进行监视和传递信息的能力。  
康斯坦丁称这些鸽子为「白色石屋的鸽子」。  
这种造物由石膏组成。

## 圣洛夫基金会 / ST. PAVLOV FOUNDATION

### 圣洛夫基金会 / St. Pavlov Foundation

人类和平安全理事会旗下的组织，负责对神秘学家和神秘物品等进行发现、收容、管理与利用的国际组织。  
*「银铃摇动的三刻以内，忠诚的伙伴们在此列队以待，活跃于世界各地的维和组织，神秘学家、神秘物品与神秘现象的收容与管理者。你可以放心地将孩子交给圣洛夫基金会，他会在这儿成为一个好孩子的。」*

### 组织架构

圣洛夫基金会是以联合委员会为核心、下设共融院等部门及其他下属机构的国际组织。下属机构有如委外合约人才管理组织等外勤组织、第一防线学校作为教育机构、负责预防和侦查神秘学家犯罪案件的夜巡特遣管理局以及拥有独立编制的司辰小队，并在各地都兼有分部办事处。
